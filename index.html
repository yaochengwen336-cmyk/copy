剪映批量复制脚本

这是一个用于剪映时间线的 PyAutoGUI 自动化脚本，会把已选中的视频片段复制并粘贴 143 次。

下载 Python 脚本 在线查看源码

运行前准备

  1. 安装 Python，并执行：pip install pyautogui
  2. 打开剪映，把视频片段放在主轨上。
  3. 用鼠标单击选中该片段，让它出现白色边框。
  4. 运行脚本后，5 秒内切回剪映窗口。

注意

  • 脚本会模拟键盘操作，请运行前确认剪映窗口处于正确状态。
  • 紧急停止：把鼠标移动到屏幕左上角，PyAutoGUI 会触发安全停止。
  • GitHub Pages 只能展示网页，不能直接在浏览器里运行这个 Python 脚本。

源码预览

import time
import pyautogui

pyautogui.FAILSAFE = True
pyautogui.PAUSE = 0.1

COPY_COUNT = 143

def main():
    print("=" * 50)
    print("剪映批量复制143次")
    print("=" * 50)
    print("脚本将在 5 秒后开始，请立即切换到剪映窗口。")

    for s in range(5, 0, -1):
        print(f"  {s}...")
        time.sleep(1)

    pyautogui.hotkey("ctrl", "c")
    time.sleep(0.5)

    for i in range(1, COPY_COUNT + 1):
        pyautogui.press("end")
        time.sleep(0.3)
        pyautogui.hotkey("ctrl", "v")
        time.sleep(0.3)

        if i % 10 == 0:
            print(f"已完成 {i}/{COPY_COUNT}")

    print(f"完成，共复制粘贴 {COPY_COUNT} 次。")

if __name__ == "__main__":
    main()
